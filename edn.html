  3. Analysis of Sudoku Game from Image Extraction Using Deep Learning (Individual Mini Thesis)       Oct 2020

    4. This project focuses on the implementation of deep learning techniques, particularly Convolutional Neural Networks (CNNs), to analyze Sudoku game boards extracted from images. The primary objective is to develop a model capable of accurately recognizing and determining handwritten digits within Sudoku grids. The workflow of this analysis involves several key stages:

    • Image Preprocessing: Initially, the input image containing the Sudoku grid is subjected to preprocessing stages using image processing methods. These stages are designed to enhance the clarity and quality of the image data, facilitating more accurate digit recognition.
    • Digit Extraction: Following preprocessing, the digits within the Sudoku grid are extracted and converted into a 2D convolution matrix. This matrix serves as the input for the CNN model. CNN Model Training: The CNN model is trained using a dataset sourced from the MNIST database. This dataset consists of thousands of handwritten digit images, which are used to train the model to recognize digits with high accuracy.
    • Digit Recognition and Game Solving: Once trained, the CNN model is deployed to recognize the digits within the Sudoku grid. Additionally, a backtracking algorithm implemented in Python is employed to compute the solution for the Sudoku game based on the identified digits.

    Through this approach, our project aims to automate the process of solving Sudoku puzzles from image inputs, offering a practical and efficient solution for Sudoku enthusiasts and developers alike.
